# Install

```
npm install
```

# Start

```
npm run dev
```

# Setup MongoDB

```
MONGO_INITDB_ROOT_USERNAME=your_username
MONGO_INITDB_ROOT_PASSWORD=your_password
MONGO_INITDB_DATABASE=your_db_name
MONGODB_USER=your_personnal_user
MONGODB_USER_PASSWORD=your_personnal_password
```

```
docker compose up -d
```
